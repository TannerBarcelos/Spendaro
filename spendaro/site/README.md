# Spendaro Marketing Site

* This is the source code for the static site hosted [here](https://tannerbarcelos.github.io/Spendaro/). It is used as a landing page for _selling_ the application
as well as a place to learn how to use Spendaro and other guides. 

* Think of this as the landing page to other popular apps like [YNAB](https://ynab.com) or [Mint](https://mint.com). These are not the apps themselves, but rather a place to learn about them and sign up for them.
You are then redirected to the actual app. 

* This site is built using Astro and deployed to GitHub Pages.
